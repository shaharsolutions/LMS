ברוכים הבאים למרכז ההדרכה

הכלים המתקדמים ביותר לניהול למידה בארגון שלך.

ניהול רב-ארגוני

המערכת תומכת במבנה Multi-tenancy המאפשר לכל ארגון או מחלקה לנהל את המידע שלהם בנפרד ובאופן מאובטח לחלוטין.

תקן SCORM בינלאומי

תמיכה מלאה בתקני SCORM 1.2 ו-2004 המאפשרים למידה אינטראקטיבית, שמירת התקדמות ומעקב ציונים מפורט.

נגישות מלאה

המערכת מותאמת באופן מלא לעבודה מכל מכשיר - מחשב, טאבלט או סמארטפון, ומאפשרת למידה מכל מקום.

מדריך למנהלי הדרכה

ניהול משתמשים, הקצאת תכנים והפקת דוחות ביצועים.

ניהול עובדים

הוספת עובדים חדשים, ניהול מחלקות ואיפוס סיסמאות בקלות דרך ממשק הניהול של הארגון.

העלאת תכנים

העלאת קבצי ZIP של לומדות. המערכת מפרקת ומאנדקסת את קבצי ה-SCORM באופן אוטומטי.

ניתוח נתונים

דוחות התקדמות בזמן אמת - מי התחיל, מי סיים, ציונים ממוצעים וזמן שהושקע בכל פרק למידה.

מדריך ללומד

כך תפיקו את המקסימום מחוויית הלמידה הדיגיטלית שלכם.

1

כניסה למערכת

התחברו עם שם המשתמש והסיסמה שקיבלתם ממנהל ההדרכה. מומלץ לשנות את הסיסמה בכניסה הראשונה.

2

לוח הלומדות שלי

במסך הראשי תמצאו את כל ההדרכות שהוקצו לכם. תוכלו לראות את אחוז ההתקדמות בכל לומדה.

3

למידה רציפה (Bookmark)

המערכת שומרת אוטומטית היכן עצרתם. גם אם תסגרו את הדפדפן באמצע, תוכלו לחזור בדיוק לאותה נקודה.

טיפים ותמיכה טכנית

עצות למידה מוצלחת ופתרון בעיות נפוצות.

טיפים ללמידה אפקטיבית

  • שימוש בדפדפן מעודכן: השתמשו ב-Chrome או Edge לחוויה מיטבית.
  • סביבת עבודה: בצעו את הלמידה במקום שקט עם חיבור אינטרנט יציב.
  • סיום הדרכה: הקפידו להגיע למסך הסגירה הסופי כדי לוודא שכל הנתונים נשלחו ל-LMS.
  • תעודות: חלק מהלומדות מאפשרות הורדת תעודת סיום לאחר השלמת 100% מהתוכן.
